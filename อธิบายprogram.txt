สวัสดีครับพี่ตองในส่วนของ web ที่ผมเขียนขึ้นมา ผมให้จำลองมันเสมือนกับเรามีพื้นที่พื้นที่หนึ่ง แล้วใช้พื้นที่ตรงนั้นเหมือนกันเป็นการให้จองร้านค้าขายของ
ซึ่งตอนนี้ผมยังไม่ได้ทำแบบล็อควันเวลาชัดเจนยังไม่มีฟังชั่นเกี่ยวกับคำนวณเรื่องเวลาเลย

program ทำอะไรได้บ้าง
USER{
    - ผู้ใช้ที่ไม่ได้ login สามารถเข้าชมเว็ปไซต์ในหน้าหลักได้ หน้าจองได้แต่จะไม่สามารถกดจองได้ 
    - login logout แยกuser Admin (userจะไม่สามารถเข้าไปในส่วนของระบบหลังบ้านได้)
    - หน้า Homepage{
        - มีbanner คือในส่วนของbannerที่จริงส่วนนี้ผมคิดไว้ว่ามันคือป้ายประกาศวันที่เราจะเปิดให้เช่าตลาดตามเทศกาลนั่นนี่ ซึ่งตรงนี้ต้องอยู่ใน
        database แต่ผมยังไม่ได้เอาลง database ฟิกไว้ทดสอบก่อนครับ //(ได้ทำการปรับเป็นดึงจากdatabaseเรียบร้อย 3/14/2025)
        - ผังพื้นที่ในส่วนนี้ผมได้ให้แฟนช่วยวาดขึ้นมาโดยเอาต้นแบบผังมาจากตลาดต้นสน (ยังไม่ได้เอาลงdatabase)
        - จะ random ตัวแผงที่ว่างให้จองขึ้นมา3แผง พอกดทีปุ่มจองแผงเลย ก็จะเด้งไปหน้าสำหรับการจองแผง
        - มีให้ รีวิว สามารถแสดงความคิดเห็นได้ ข้อมูลส่วนนี้ บันทึกลง database เรียบร้อย แล้วดึงคอมเม้นมาแสดง
        - คำถามที่พบบ่อย
        - สามารถถามคำถามหรือแจ้งปัญหาได้
        
    }
    - user สามารถแก้ไขข้อมูลตัวเองที่สมัครมาได้ เปลี่ยนรหัสผ่านได้
    - หน้าสำหรับการจอง เมื่อมีuserจองแล้ว จะไม่สามารถจองที่นั้นอีกได้
    - หน้าแผงของฉันเอาไว้ให้userที่จองเสร็จไปกดเพื่ออัพสลีป ให้ทาง Admin ได้ตรวจสอบ
    - ดูร้านค้าต่างๆที่จองไว้โดยจะไม่แสดงข้อมูลที่ละเมิดความเป็นส่วนตัวของ user 
    - userสามารถที่จะยกเลิกจองเองได้(เฉพาะพื้นที่ที่ยังไม่ได้มีการอัพสลีป)

}
Admin{
    - จัดการผู้ใช้งาน แก้ไขข้อมูล / ลบ
    - จัดการรายการจองตลาด ไว้เคลียกดยกเลิกจองเพื่อให้หน้าจองของuserขึ้นว่างให้จองได้
    - เพิ่มแผงตลาด
    - แก้ไขข้อมูลแผงตลาด แก้ไข / ลบ
    - ตรวจสอบการชำระเงินเมื่อผู้ใช้อัพสลีปมา
    - ตรวจเช็คการชำระเงินทั้งหมด
    - Create admin
    - อัพโหลด banner
    - จัดการแก้ไข ลบ banner
    - จัดการ banner สามารถอัพโหลดได้ แก้ไขได้ ลบได้ และเลือกภาพที่จะแสดงขึ้นwebได้
    - จัดการ รีวิว ที่user รีวิวเข้ามา
    - add คำถามที่พบบ่อย รวมถึงหน้าเอาไว้ จัดการ แก้ไข/ลบ
    - ตอบคำถาม/ปัญหา ที่ userถามเข้ามา
}
